# Licensing and Provenance Policy

## Original Bahrain Brick work

Original documents, specifications, procedural scripts and concept sheets in this repository are project-owned work prepared for Muhamed Zanabal. No public license grant is implied. A root distribution license must be selected before public release.

## Third-party acceptance

Accepted by default:

- CC0
- CC BY with complete attribution and redistribution compliance
- MIT/BSD/Apache-2.0 for code or shaders with notices preserved

Rejected by default:

- noncommercial or personal-use-only terms
- unclear or missing license
- ripped commercial-game content
- marketplace assets whose redistribution or cross-engine use is not clearly permitted
- copied maps, scans, CAD or landmark geometry
- assets with unverifiable AI-generation rights

## Existing source repository status

The inherited asset tree is quarantined. The source audit reported 321 rows, of which 11 were blocked and 310 required provenance. Flexible Toon Shader upstream is MIT, but the exact bundled Godot 4 derivative provenance still requires review. No inherited binary is copied here.
