# Bahrain Brick Asset Production

Isolated source-controlled asset-production repository for **Bahrain Brick**.

This repository is intentionally separate from `MuhamedZanabal/brick-bahrain-open-world`. The game repository is treated as read-only evidence. No protected controls, product source, signing secrets, APKs, or unverified third-party binaries are copied here.

## Current scope

- authority and licensing audit
- controlled art direction and Android budgets
- manifest-driven asset pipeline
- first architecture production batch: villa, Muharraq-style traditional, Manama souq, and Bahrain Bay-inspired waterfront
- deterministic SVG technical concept sheets
- Blender procedural starter generators
- isolated Godot 4.3 asset-gallery and benchmark scaffolding
- validation tooling and CI specification

## Branch policy

- `main`: repository bootstrap only
- `work/bahrain-brick-complete-asset-system-v1`: active asset-production authority

A complete local Git repository, Git bundle, and ZIP are therefore produced. See `evidence/REPOSITORY_CREATION_BLOCKER.md` and `scripts/publish_to_github.sh`.
